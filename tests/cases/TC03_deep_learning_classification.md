# 1. 测试用例名称

基于深度学习的用户行为分类功能（黑盒-打包后EXE）

# 2. 测试用例综述

验证分类模型可对会话/窗口片段进行本人/非本人/非典型行为分类，并输出分类结果与置信度。

# 3. 用例初始化

- 准备带标签的数据集（训练/验证分开），或加载随 EXE 附带的已训练模型。
- 若支持：通过参数指定评估模式（例：`--mode eval-cls --data <path>`）。

# 4. 前提和约束

- 依赖的深度学习框架与加速环境可用（若含 GPU）。
- 数据标签可靠且类别分布可接受（必要时加权/重采样）。
