# 1. 测试用例名称

用户行为分类算法准确率指标（黑盒-打包后EXE）

# 2. 测试用例综述

验证基于深度学习的分类算法在验证集上的准确率 ≥ 90%，F1-score ≥ 85%。

# 3. 用例初始化

- 准备带标签的验证集（划分独立于训练数据）。
- 运行 EXE 的评估流程，产出分类报告（JSON/CSV/日志）。

# 4. 前提和约束

- 类别分布不均衡时使用加权或宏/微平均 F1。
- 保证随机种子与评估配置可复现。
